# Gossips Admin

Next.js admin dashboard for the Gossips / Workplace Buzz project. Connects to the [gossips_backend](https://github.com/musamaGamma/gossips_backend) API.

## Setup

```bash
npm install
# Configure .env.local with API URL and admin credentials
npm run dev
```

## Scripts

- `npm run dev` – development server
- `npm run build` – production build
- `npm run start` – run production build
